# AnalystLab Africa Data Analytics Internship – Week 1 & Week 2

## Project Title

Data Cleaning and Exploratory Data Analysis (EDA)

## Internship

**AnalystLab Africa – Data Analytics Internship Program**

## Objective

The objective of this project was to learn the fundamentals of data analytics by cleaning raw datasets, exploring the data, identifying patterns, and presenting meaningful insights using Python.

## Datasets Used

### 1. Online Retail Dataset

* Sales transaction data from an online retail store.
* Used to analyze customer purchases, product sales, and order patterns.

### 2. Netflix Movies and TV Shows Dataset

* Information about Netflix titles, including type, country, release year, rating, and genre.
* Used to analyze content distribution and trends.

## Tools & Technologies

* Python
* Jupyter Notebook / VS Code
* Pandas

## Tasks Performed

* Imported datasets into Python.
* Checked dataset structure and data types.
* Identified and handled missing values.
* Removed duplicate records.
* Cleaned inconsistent data.
* Generated summary statistics.
* Performed Exploratory Data Analysis (EDA).

## Key Insights

### Online Retail Dataset

* Sales were concentrated in a small number of popular products.
* Some transactions contained negative quantities, indicating returns or cancellations.
* Customer purchasing behavior varied across countries.
* Most transactions had relatively small order quantities.

### Netflix Dataset

* Movies made up a larger share of the catalog than TV Shows.
* The number of titles increased significantly over recent years.
* Content was distributed across many countries and genres.
* Certain ratings appeared more frequently than others.

## Skills Learned

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Descriptive Statistics
* Python Programming
* Business Insight Generation

## Conclusion

This project strengthened my understanding of the complete data analysis workflow, from cleaning raw data to generating business insights through visualizations. It provided practical experience with Python libraries commonly used by data analysts.
